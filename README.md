GitHub Graveyard is a web app for discovering forgotten open-source repositories, tagging their status, and supporting projects with “candles”.

## Status

This project is **still in development**, but the app is already live:

- https://gitgrave.com/

## Features

- Find projects you care about and quickly read GitHub signals (stars, issues, activity).
- Paste a repo link and get a quick “story snapshot”.
- Tag the project: abandoned / finished / underrated.
- Explore **Category Worlds** — a visual space where tagged projects appear by status.
- Light candles to boost a project and help it get noticed.

## Stack

- Next.js 14 (App Router)
- PostgreSQL
- Prisma
- next-intl
- Tailwind CSS
